总结

pwntools在密码学交互中经常会用到，之后大概会专门找几道例题来讲解该功能，在这里简单介绍几个函数：

  1. interactive() : 直接进行交互，相当于回到shell的模式，在取得shell之后使用
  2. recv(numb=4096, timeout=default) : 接收指定字节
  3. recvline(keepends=True) : 接收一行，keepends为是否保留行尾的\n
  4. recvuntil(delims, drop=False) : 一直读到delims的pattern出现为止
  5. send(data) : 发送数据
  6. sendline(data) : 发送一行数据，相当于在数据末尾加\n
